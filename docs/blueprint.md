# **App Name**: AI Assistant Hub

## Core Features:

- Chatbot Dashboard: Display a list of all created chatbots with key information such as name, status, last updated, and usage statistics.
- New Chatbot Workflow: Guide users through a multi-step process to create a new chatbot, including naming, attaching knowledge sources, defining persona & rules, and selecting a deployment method.
- Centralized Data Source Library: Allow users to upload and manage documents (PDFs, DOCX, etc.) and website links in a central repository for use across multiple chatbots.
- AI Data Processing Tool: Use AI to process and tag uploaded documents and index website links, categorizing them into the knowledge library.
- Chatbot Personality Builder Tool: Let users set the personality and core rules for each chatbot, and Gemini will use this tool in crafting responses.
- Embeddable Chat Widget: Generate code snippets for embedding chatbots on websites.
- Chatbot Direct Link: Create direct URLs for chatbots to share on social media or in emails.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional, trustworthy feel.
- Background color: Light gray (#F5F5F5), providing a clean, uncluttered backdrop.
- Accent color: Complementary orange (#FF9800) for call-to-action buttons and key highlights, providing contrast and guiding user attention.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a clean, modern aesthetic, ideal for both headings and body text.
- Use clear and concise icons to represent different chatbot functions and data sources.
- Implement a clean and intuitive layout, prioritizing easy navigation and information accessibility. The chatbot dashboard should be the focal point, with quick access to essential functions.
- Employ subtle animations to enhance user experience, such as loading indicators and transitions between chatbot management sections.
